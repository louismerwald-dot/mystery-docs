# Background music

Drop a royalty-free MP3 here named `bgm.mp3` to mix under the narration at 8% volume.

For mystery content, pick **dark ambient / cinematic underscore** tracks. Avoid:
- Anything melodic or with hooks (fights the storytelling)
- Bright/upbeat (kills the mood)
- Anything with vocals
- Horror-movie stingers (too aggressive for documentary)

Free sources with good dark-ambient selections:
- **Pixabay Music** (search "dark ambient", "cinematic underscore", "tension"): https://pixabay.com/music/
- **YouTube Audio Library** (filter by genre: Cinematic, mood: Dark): https://studio.youtube.com (Audio Library)
- **Free Music Archive** (Cinematic / Dark Ambient genres): https://freemusicarchive.org

Recommended track length: 2-3 minutes. The pipeline loops it automatically.

If no `bgm.mp3` is present, narration plays solo (still works, just less atmospheric).
